# gmap-vue-example

Sample vue web application to create google maps with [gmap-vue]() plug-in

```
yarn
yarn serve
```

## Setup

create `.env` file for configuration

```
VUE_APP_GMAP_KEY=<YOUR_GOOGLE_MAPS_API_KEY>
```
